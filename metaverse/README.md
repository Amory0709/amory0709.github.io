# 🌐 SLB 招聘元宇宙体验

一个基于 Three.js 的 3D 招聘元宇宙场景，让应聘者通过互动游戏了解企业文化。

## 🎮 在线体验

**立即体验：** [https://amory0709.github.io/metaverse/](https://amory0709.github.io/metaverse/)

## 📖 项目简介

这是一个创新的招聘工具，将传统的招聘流程转化为沉浸式的 3D 体验：

- 🎯 **互动游戏** - 通过收集信息方块了解公司文化
- 🏢 **3D 场景** - 逼真的办公环境和建筑
- 📊 **实时反馈** - 即时的得分和进度显示
- 🎵 **音效体验** - 沉浸式的声音设计
- 📱 **响应式设计** - 支持桌面端和移动端

## 🎯 游戏玩法

1. **开始探索** - 点击"开始探索"按钮
2. **控制角色** - 使用 WASD 键移动，鼠标控制视角
3. **收集信息** - 靠近蓝色方块按 E 键收集公司信息
4. **查看展示** - 与展示板交互了解更多详情
5. **完成挑战** - 收集所有方块获得高分

## 🔧 技术特点

- **Three.js** - 3D 图形渲染
- **原生 JavaScript** - 无框架依赖
- **WebGL** - 硬件加速渲染
- **Web Audio API** - 音效系统
- **响应式设计** - 自适应不同设备
